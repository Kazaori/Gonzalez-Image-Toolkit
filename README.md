# 数字图像处理实验系统 (DIP Lab System)

基于 冈萨雷斯《数字图像处理（第四版）》核心理论开发的专业级图像处理教学与实验软件。
采用 Python + PyQt5 构建，底层算法依托 OpenCV、NumPy 与 SciPy，辅以 Matplotlib 和 Pillow 进行数据分析与高质量导出。

---

## 1. 实验目的与项目简介

本系统旨在为《数字图像处理》课程提供一个直观、严谨且功能完备的实验平台。
传统的代码脚本调试往往缺乏直观的参数对比，而现有的商业软件（如 PS）又封装得过于黑盒。本软件**完全按照教材的章节脉络**设计，将晦涩的数学公式转化为可视化的交互界面，帮助学习者深入理解空间域与频率域的图像处理机制。

**主要功能覆盖：**
* **第 2 章**：空间几何变换（仿射变换、镜像映射）。
* **第 3 章**：灰度变换（线性/非线性伽马）、直方图处理（全局/CLAHE）、空间域滤波（平滑/中值去噪）。
* **第 4 章**：频率域滤波（理想/高斯/巴特沃斯 低通与高通滤波器）。
* **第 5 章**：图像复原（常见退化噪声模型的引入）。

确保您的计算机已安装 Python 3.7+ 环境。

## 3. 核心使用逻辑 (重要操作指南)

本系统采用了**“流水线（Pipeline）与非破坏性对比”**的交互逻辑，高度契合科学实验控制变量的原则：

1. **双图对比视口**：
   * **左侧（当前原图）**：作为算法处理的**基准输入（Base Image）**。
   * **右侧（处理结果）**：实时展示当前参数下的**输出结果（Processed Image）**。
   * *视口联动*：两侧画布支持鼠标拖拽平移与同步缩放，方便在像素级对比处理前后的细节差异。
2. **流水线串联 (📌 固化为原图)**：
   * 所有的右侧选项卡操作，默认**只对左侧的原图生效**（避免多次点击按钮导致效果无序叠加）。
   * 如果你需要进行**多步连续处理**（例如：先加噪声 $\rightarrow$ 设为原图 $\rightarrow$ 再进行滤波去噪），请点击顶部工具栏的 **[📌 固化为原图 (进入下一步)]**，右侧结果将被锁定至左侧，进入下一处理环节。
3. **安全撤销机制**：
   * 系统内置历史状态栈。随时点击 **[⬅️ 撤销 (回到上一步)]** 退回上一步，或点击 **[⏮️ 回到初始图像]** 瞬间重置回刚载入的最初状态。
4. **高质量学术导出**：
   * 点击 **[💾 保存结果]** 时，系统会弹出 DPI（分辨率）与质量设置窗口。建议保留默认的 `300 DPI` 导出 TIFF 或 高质量 JPG，以便直接插入学术论文或实验报告排版。

## 4. 功能模块与课本理论映射 (实验原理)

### 4.1 亮度增强与直方图处理 (对应教材第 3 章)
* **直方图均衡化 (Histogram Equalization)**：
  通过计算图像灰度的累积分布函数 (CDF) 进行灰度映射，使输出图像的概率密度函数 (PDF) 趋于均匀，从而实现全局对比度的自适应最大化。系统同时提供了 **CLAHE（限制对比度的自适应直方图均衡化）**，能有效克服全局均衡化在局部区域导致的过曝和噪点放大问题。
* **伽马校正 (Gamma Correction)**：
  * **公式**：$s = c \cdot r^\gamma$
  * **原理**：基于查找表（LUT）实现非线性映射。
  * **分析**：当 $\gamma < 1$ 时，拉伸低灰度级，使**整体画面提亮**（显现暗部细节）；当 $\gamma > 1$ 时，压缩低灰度级，使**整体画面压暗**。

### 4.2 几何空间变换 (对应教材第 2 章)
* **仿射变换矩阵**：
  融合了旋转、缩放与平移的二维空间映射。软件底层使用 OpenCV 的 `warpAffine` 计算。
* **边界处理**：
  当图像旋转或缩小后，不可避免会产生未定义区域。本系统采用零值（纯黑）填充（`BORDER_CONSTANT`），严格避免了错误拉丝外推产生的视觉伪影。

### 4.3 图像加噪与空间域滤波 (对应教材第 3、5 章)
* **噪声模型引入**：
  * **高斯噪声 (Gaussian)**：基于正态分布，主要由传感器热噪声引起。调整“方差”可控制其严重程度。
  * **椒盐噪声 (Salt & Pepper)**：基于概率的脉冲噪声（黑白极值点），模拟电路突变。
* **空间域去噪**：
  * **均值滤波**：线性空间滤波，会导致边缘变模糊。
  * **中值滤波**：非线性统计排序滤波。**（实验验证要点：对比发现，中值滤波在消除椒盐噪声时表现完美，且能极好地保护图像边缘锐度）**。

### 4.4 频率域滤波 (对应教材第 4 章)
* **傅里叶变换基础**：
  底层通过 `np.fft.fft2` 进行二维离散傅里叶变换，并使用 `fftshift` 将低频分量移动至频谱中心。
* **滤波器设计**：
  * **理想滤波器 (Ideal)**：截断极其生硬，逆变换回空间域后会产生强烈的**振铃效应 (Ringing Effect)**。
  * **高斯滤波器 (Gaussian)**：由于高斯函数的傅里叶逆变换仍是高斯函数，因此**完全没有振铃效应**，过滤平滑自然。
  * **巴特沃斯滤波器 (Butterworth)**：介于两者之间，通过引入“阶数 $n$”来控制频率截断的陡峭程度。
  * **低通(Lowpass) vs 高通(Highpass)**：低通保留背景和模糊轮廓；高通滤除背景，提取尖锐的边缘细节。

## 5. 实验分析辅助工具

* **全彩 2x2 直方图矩阵**：
  点击左下角【并排显示直方图面板】，如果输入的是彩色图像，系统将使用 Matplotlib 自动绘制 **2 行 2 列** 的分析面板：
  * **上排**：处理前后的单通道黑白明度直方图对比。
  * **下排**：处理前后的 R、G、B 三通道曲线分布对比。
  * *用途*：非常适合在报告中论证“直方图均衡化如何将集中在低亮度区间的像素均匀拉伸至整个 0-255 空间”。
